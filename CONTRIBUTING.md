# Contributing to ToolChainEnv

We welcome contributions to `tool_chain_env`!

## Development Setup

1.  Clone the repository.
2.  It is recommended to use a Python virtual environment.
3.  Install the dependencies: `pip install -r requirements.txt`
4.  Install the development dependencies: `pip install -r requirements-dev.txt` (You may need to create this file if it doesn't exist and add packages like `pytest`).

## Running Tests

Run the test suite with `pytest`.

```bash
pytest
```

## Code Style

Please adhere to the `black` code style.

## Submitting a Pull Request

1.  Fork the repository.
2.  Create a new branch for your feature.
3.  Add your feature or bug fix.
4.  Add tests for your changes.
5.  Ensure all tests pass.
6.  Submit a pull request.
